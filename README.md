# Bigram - Shakespeare 文本分析

基于大数据分析的莎士比亚文集文本处理项目。

## 项目简介

对 Shakespeare 文集进行文本分析：
- Bigram 词组统计
- 关键词搜索（如 torture）
- 文本处理与统计

## 技术栈

- **语言**: Java
- **框架**: Hadoop MapReduce / Spark

## 功能特性

- ✅ Bigram 统计分析
- ✅ 关键词搜索
- ✅ 文本清洗处理
- ✅ 词频排序

## 项目说明

列出每一行含有 "torture" 的内容，以及统计 bigram（双词组合）频率。
